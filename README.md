# My Skills

A repository of common skills and Cursor rules that I use in my projects.

## Getting Started

You can install these skills with the [`skills`](https://github.com/vercel-labs/skills) CLI.

```console
npx skills add Michael-F-Bryan/skills --all
```

Sometimes you'll tell the `skills` CLI to use symlinks, but it will copy the files into your project anyway. An approach that is useful when developing locally is to symlink the skills to your global agents folder.

After cloning this repository, run the following:

```console
mkdir -p ~/.agent/skills
ln -sf skills/* ~/.agent/skills
```
