# Rich HTML channel

Rich HTML artefacts (decision briefs, explainers, review aids) need **two skills**:

| Concern | Skill |
|---|---|
| Workflow, React scaffold, templates, component patterns, verification | [html-rich-communication](../../html-rich-communication/SKILL.md) |
| Operator Notes colours, typography, web-base CSS | michael-brand-pack (this skill) |
| Writing voice and artefact tone | [voice-michael](../../voice-michael/SKILL.md) |

## What to copy from this skill

When html-rich (or its init script) integrates an attached brand skill:

- `assets/css/tokens.css`
- `assets/css/web-base.css`

HTML layout components and templates live in html-rich — not here.

## Start here

1. Follow html-rich workflow (artefact brief → content brief → visual plan → scaffold → bundle → browser review).
2. Attach this skill for tokens and web-base CSS.
3. Attach voice-michael for tone and structure.

Do not duplicate html-rich workflow steps in brand references.
