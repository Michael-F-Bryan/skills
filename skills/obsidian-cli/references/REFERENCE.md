# Obsidian CLI References

Read supporting files only when the task needs them.

## Choose a file

- Read [commands.md](commands.md) when you need command discovery, command families, or a write-verification checklist.
- Read [coding-workflows.md](coding-workflows.md) when you need to choose the right note type or record coding context in the vault.
- Read [voice.md](voice.md) when you need help drafting concise note prose in the preferred style.

## Default order

1. Start with `SKILL.md`.
2. Read `commands.md` if the command surface or safe mutation flow is unclear.
3. Read `coding-workflows.md` if the task is about note choice, handover, daily notes, or decision capture.
4. Read `voice.md` only if you need writing-style guidance.
